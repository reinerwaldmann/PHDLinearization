===Линеаризация===
1. Функция, делающая производную
вх - последовательность функций (строк)
список переменых, по которым брать производные 
номер производной 
вых - список производных в виде строк
Оные производные записываются в глобальные переменные, после чего 
поднимается соответствующий флаг. При поднятом флаге производные не считаются заново, 
а возвращаются из глобальных значений.
Ещё вариант - они скидываются в файл, называемый derivate.py
Где формируется функция, делающая производные.
В этом файле найдутся глобальные переменные, которые равны функциям, поданным на вход (строки)
Теперь алгоритм
Функция производной получает на вход некоторые функции, из которых надо найти производные

2. Функция, которая считает производные 
вх последовательность функций (строк), словарь переменная -
значение, список переменных, по которым надо брать производные.
выход - список значений производных
Функцию (1) запускает сама

3. Функция, которая считает дисперсию линеаризацией
 вх - последовательность строк-функций 
словарь значений переменных (всех)
список переменных, по которым надлежит брать производные
ковариационная матрица V
выход - последовательность дисперсий по каждой функции (возможно, словарь)
Запускает (2)
