S>Добрый день!

S>Нужна случайная последовательность, скажем 1000 чисел, с заданными мат. ожиданием, дисперсией и корреляционным коэффициентом между соседними значениями. Ну, или в общем случае с заданной ковариационной матрицей (если коррелированы не только соседние значения). Как такое можно сделать?

S>Спасибо)

Пусть тебе надо ряд из t величин с матожиданием (m1,m2,...,mt) и ковариационной матрицей V[t x t] (она должна быть неотрицательно определенной и симметричной).

Генеришь ряд X=(X1,X2,...,Xt) из независимых случайных с нулевым матожиданием и единичной дисперсией. Находишь матрицу W[t x t] такую, что W'W=V (называется это разложением Холецкого), и берешь Y = XW + (m1,m2,...,mt).

В случае, если тебе надо, чтобы только соседние были скоррелированы, т.е. матрица V имеет трехдиагональный вид, то матрица W будет двухдиагональной, причем ее колонки можно вычислить последовательно (в первой колонке будет всего один ненулевой элемент -- корень из дисперсии первой случайной величины, а во всех остальных будет по два ненулевых элемента, т.е. при преобразовании, начиная со второго элемента, надо всегда брать взвешанную сумму предыдущего и текущего элемента), а это значит, что можно за линейное время преобразовать ряд X в ряд Y.

Пример: надо получить ряд, матожидание равно 0, дисперсия всех равна 1, ненулевая корреляция только соседних элементов ряда и должна быть равна 0.5. Тогда генерим ряд независимых равномерно распределенных случайных на [-sqrt(3),sqrt(3)], все они имеют нулевое матожидание, единичную дисперсию, а затем домножаем этот ряд на матрицу W такую, что
W'W=
| 1   0.5 0   0   ... |
| 0.5 1   0.5 0   ... |
| 0   0.5 1   0.5 ... |
| ... ... ... ... ...

т.е.
W=
1.0     0.5     0       0
0       0.86603 0.57735 0
0       0       0.81650 0.61237
0       0       0       0.79057

т.е.
Y1=X1
Y2=0.5X1+0.86603X2
...
А вот зайца кому, зайца-выбегайца?!
